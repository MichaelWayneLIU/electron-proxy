# proxy

**基于electron开发的内网穿透工具.**

## 使用说明

按以下步骤：

```bash
# Clone this repository
git clone https://github.com/try-to/proxy
# Go into the repository
cd proxy
# Install dependencies
npm install
# Run the app
npm start
```
速度可能会很慢，是因为网络原因，建议使用[淘宝 NPM 镜像](https://npm.taobao.org/).
