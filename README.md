# proxy

**基于electron开发的内网穿透工具.**

## 使用说明

下载之后，按一下操作

```bash
# Clone this repository
git clone https://github.com/try-to/proxy
# Go into the repository
cd proxy
# Install dependencies
npm install
# Run the app
npm start
```
